# 🛹 Vans Website Clone

Este é um projeto de recriação do site da **Vans**, desenvolvido com o objetivo de praticar habilidades de front-end e aprimorar meus conhecimentos em desenvolvimento web.

⚠️ Este projeto ainda está em desenvolvimento e não está finalizado.

---

## 🚀 Sobre o projeto

O objetivo deste projeto é recriar a interface do site da Vans, focando no design, estrutura e responsividade.

Ele foi criado como forma de prática para melhorar minhas habilidades em:
- Estruturação de páginas web
- Estilização com CSS
- Organização de layout
- Boas práticas de desenvolvimento

---

## 💻 Tecnologias utilizadas

- HTML5  
- CSS3
